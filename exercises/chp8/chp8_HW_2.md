| No. | 관계 타입 | 관련된 개체 목록 | 관계 유형 | 관계 설명 속성       |
| --: | ----- | --------- | ----- | -------------- |
|   1 | 지도    | 교수, 학생    | 1 : N | 없음             |
|   2 | 강의    | 교수, 교과목   | M : N | 강의년도, 강의학기, 분반 |
|   3 | 수강    | 학생, 교과목   | M : N | 수강년도, 수강학기, 성적 |


3.

교수(사번 PK, 직급, 전공, 입사일)

학생(학번 PK, 이름, 주소, 학년, 지도교수사번 FK)

교과목(교과목번호 PK, 교과목명, 학점, 구분)

강의(
    사번 FK,
    교과목번호 FK,
    강의년도,
    강의학기,
    분반,
    PRIMARY KEY(교과목번호, 강의년도, 강의학기, 분반)
)

수강(
    학번 FK,
    교과목번호,
    수강년도,
    수강학기,
    분반,
    성적,
    PRIMARY KEY(학번, 교과목번호, 수강년도, 수강학기, 분반),
    FOREIGN KEY(교과목번호, 수강년도, 수강학기, 분반)
        REFERENCES 강의(교과목번호, 강의년도, 강의학기, 분반)
)
